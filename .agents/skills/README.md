# スキル (Skills)

スキル機能は、特定の特化タスクについてAIエージェントの能力を拡張するための指示やスクリプトなどのリソースをまとめたフォルダです。
プロジェクト特有の開発ルール、アーキテクチャのお作法、レビューの観点などを登録しておくことで、エージェントがより高い精度でサポートできるようになります。

## スキルの作り方

プロジェクトのルートディレクトリに `.agents/skills/<スキル名>/` フォルダを作成し、その中に `SKILL.md` を配置します。

### `SKILL.md` のフォーマット

```yaml
---
name: スキル名
description: このスキルが何をするか、どういった時に使うかの説明
---
```

その下にマークダウン形式で詳細な指示を記載します。

## 例:

- コーディング規約を教えるスキル
- 新規ページ追加時の定型作業を自動化するスキル
- 特定のライブラリに関するお作法を教えるスキル
